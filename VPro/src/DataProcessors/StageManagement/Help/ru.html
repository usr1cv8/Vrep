АРМ "Выполнение этапов производства"

Рабочее место позволяет организовать массовое формирование производственных документов на основании заказов покупателей и на производство. Продукция может производиться в несколько этапов разными датами и на разных подразделениях (при включении опции "Этапы могут выполняться разными подразделениями" в настройках программы). Порядок этапов указывается для каждой спецификации отдельно заполнением реквизита "Вид производства" (изменить виды производства). Каждый материал или операция привязываются к определенному этапу.

В АРМ отображаются все незакрытые заказы покупателей и заказы на производство, проведенные после включения опции "Использовать производство в несколько этапов" настроек программы. Для продукции, не использующей производство с разделением этапов, выводится только один предопределенный этап "Завершение производства". При использовании многопередельного производства требуется предварительное формирование заказов на производство для всех вложенных сборок (полуфабрикатов).

В правой части формы размещена скрываемая панель отборов. В группе "Формировать" указывается дата производства, которой выполняется добавление и изменение производственных документов. В группе "Заполнить" используется для подстановки общих изготовителей и исполнителей для всех выполненных этапов. Склады списания материалов и хранения готовой продукции устанавливаются в настройках автоперемещения подразделений. Также есть возможность массовой отметки выполненным определенного этапа. Остальные поля позволяют наложить фильтры на список заказов/продукции.

В процессе отметки выполненных этапов производства может потребоваться ввод дополнительной информации:

  • Изготовитель заполняется автоматически из данных заказа на производство. Если заказы на производство не используются - это поле следует заполнить вручную.
  • При использовании техопераций для этапов, требующих создания сдельных нарядов, нужно указать в отдельной колонке исполнителя
  • После включения настройки программы "Этапы могут выполняться разными подразделениями" для каждого этапа требуется указать подразделение-изготовитель. При отметке автоматически будут подставляться структурные единицы, выбранные основными в карточке этапа производства.

После нажатия на кнопку "Выполнить этапы" будут сформированы документы по отмеченным этапам и удалена информация по отмененным.

Планировщик ресурсов/Журнал записи

Планировщик ресурсов позволяет проанализировать календарный план выполнения работ, оказания услуг, выпуска продукции и полуфабрикатов на заданный период (день, неделю, месяц).

План формируется по документам Событие(с видом "Запись"), заказам-нарядам и/или заказам на производство (если для регистрации работ и услуг используется заказ покупателя, то для размещения их в плане используется заказ на производство). 

 

Определения:

 

Шаг планирования(задается в справочнике "Ресурсы предприятия").  - фиксированный интервал в который производится загрузка ресурса.

Мощность(задается в справочнике "Ресурсы предприятия").  - максимально доступная загрузка ресурса в рамках шага планирования.

Пример: Учебный класс(Ресурс предприятия). Длительность занятий - 45 мин(шаг планирования). Число мест 30(мощность).

Колонка "Итоги":

1. Занято. Отображает загрузку ресурса в выбранный период.

2. Всего. Отображает возможную загрузку в выбранный период.

 

Режимы отображения:

 

1. Шаг планирования. Отображает план-график по дням в разрезе шага планирования. Максимальный интервал отображения - 14 дней. В данном режиме доступно планирование в рамках шага планирования.

Значения интервалов:

1. Верхнее значение. Отображает максимальную сумму пересекающихся интервалов документов в рамках интервала шага планирования

Пример: Шаг планирования = 35 мин. Начало интервала 06:00, окончание 06:35. Загрузка по каждому документу = 1. Документы в интервале - (06:00 - 06:10), (06:15-06:25), (06:20-06:30). Значение интервала = (06:15-06:25) + (06:20-06:30) = 2.

2. Нижнее значение. Отображает начало интервала планирования.

 

2. День. Отображает план-график по дням в целом. Максимальный интервал отображения не ограничен. В данном режиме доступно планирование по дням.

Значения интервалов:

1. Верхнее значение по дню. Отображает итоги по каждому дню из интервала. Левое значение - отображает загрузку ресурса за день.  Правое значение - тображает возможную загрузку за день.

Пример: 40|34. Загрузка ресурса за день 40 из 34 возможных.(Превышение доступной загрузки)

2. Нижнее значение по дню. Левое значение - отображает количество занятых(!) интервалов по графику работы согласно шагу планирования. Правое значение -  отображает количество свободных интервалов по графику работы согласно шагу планирования.

Пример: 7|10. В течении дня имеются значения загрузки в 7 интервалах из 10 доступных по графику работы ресурса.

 

2. Месяц. Отображает план-график по месяцам. Максимальный интервал отображения не ограничен. В данном режиме планирование не доступно.

Значения интервалов:

В левой колнке отображается сумма загрузки ресурса за месяц. В правой колонке отображается возможная загрузка за месяц.

 

Параметры отображения устанавливается на вкладке "Настройки".

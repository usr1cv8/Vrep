Документ "Расходы при импорте товаров"

Документ предназначен для отражения таможенных сборов и пошлин, зафиксированных при поступлении импортных товаров в грузовой таможенной декларации (ГТД).

При вводе документа на закладке Главное нужно указать следующие реквизиты:

  • Таможня - заполняется наименование таможни из справочника Контрагенты,
  • Номер ГТД - номер грузовой таможенной декларации, по которой необходимо ввести информацию о таможенных пошлинах и сборах,
  • Таможенные сборы и штрафы могут быть оформлены в любой валюте. Валюта документа изменяется по гиперссылке Цены и валюта,
  • Валютный депозит и Депозит на таможне (руб) - указываются договоры с таможней, по которым учитывается взимание соответственно валютных и рублевых таможенных платежей.


Здесь же необходимо заполнить сведения о таможенных сборах и платежах, а также счета учета расчетов с таможней для отражения санкций таможни и аналитику учета расходов по штрафам.

На закладке Разделы ГТД указывается сумма таможенной пошлины, сумма взимаемого НДС, а также список товаров по каждому разделу ГТД.

Грузовая таможенная декларация может иметь несколько разделов, в которые группируются товары с одинаковым порядком расчета таможенных платежей. Для ввода информации отдельно по каждому из разделов предназначены две связанные табличные части. В верхней таблице отображается сводная информация о таможенных пошлинах и сборах по всем разделам ГТД. При установке курсора на одну из строк этой таблицы в табличной части "Товары по разделу..." отображается информация о товарах по данному разделу ГТД.

  • Таможенная стоимость - вводится в валюте документа.
  • Ставка пошлины, Ставка НДС - указывается ставка пошлины и ставка НДС.
  • Сумма пошлины - сумма таможенной пошлины вычисляется, исходя из введенной таможенной стоимости товаров и ставки пошлины. 
  • Сумма НДС - сумма взимаемого НДС.
  • Табличные части закладки Разделы ГТД заполняются информацией о товарах в валюте документа.
    • Список номенклатуры можно заполнить автоматически из документа Приходная накладная. По кнопке Изменить выбирается документ поступления, из которого весь перечень товаров попадает в табличную часть соответствующего раздела ГТД. Товары, которые не относятся к данному разделу, можно удалить, после чего повторить процедуру для следующего раздела ГТД.
    • Введенная информация о таможенной пошлине при необходимости может быть скорректирована.
    • Добавлять и удалять разделы ГТД можно по кнопке Разделы ГТД.

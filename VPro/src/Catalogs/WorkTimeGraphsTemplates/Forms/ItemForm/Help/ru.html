Шаблон заполнения графика работы.

Справочник используется для создания шаблонов заполнения справочника "Графики работы".

Возможны два варианта заполнения:

1. По дням недели. Заполнение графика привязано к дням недели, вне зависимости от периодичности дат календаря. По умолчанию шаблон заполняется пятидневкой.

2. Сменный(Произвольный). Вариант заполнения при котором необходимо указать периодичность рабочих дней(дней цикла). Не привязан к дням недели и праздникам.

В разделе "Дополнительно" вводятся дополнительные настройки заполнения графика.

Заполнение интервалов перерывов не является обязательным.

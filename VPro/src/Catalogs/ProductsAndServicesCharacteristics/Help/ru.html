Справочник "Характеристики номенклатуры"

Справочник содержит информацию о дополнительных характеристики элементов номенклатуры: цвет, размер и т.п., в разрезе которых можно вести учет запасов.

Использование справочника зависит от параметра учета системы "Вести учет по характеристикам". Значение параметра устанавливается в разделе "Настройки" - "Настройка учета".

Редактирование характеристик номенклатуры осуществляется из формы списка справочника, который открывается из формы элемента справочника "Номенклатура" или "Номенклатурная группа".

Характеристики назначаются для позиций у которых установлен флажок "Вести учет по характеристикам" на вкладке панели навигации  - "Характеристики" справочника номенклатуры. Для контроля обязательного заполнения характеристики в документах, необходимо установить флажок "Обязательное заполнение характеристик".

Для установки основной характеристики элемента справочника номенклатура/категории номенклатуры, необходимо в списке выделить нужную характеристику и нажать на кнопку "Использовать как основную".

Описания позиции номенклатуры состоят из списка значений определенных видов свойств. Причем для разных групп номенклатуры могут быть определены различные виды свойств.

Например, для описания характеристик товаров группы «Обувь» могут быть использованы такие виды свойств, как размер, цвет, полнота. Для описания товаров группы «Холодильники» используются такие виды свойств, как габариты, объем морозильной камеры, цвет отделки.

Виды свойств задаются в плане видов характеристик "Набор дополнительных реквизитов и сведений".

 

При подборе позиций номенклатуры в документ показываются остатки запасов в разрезе различных характеристик. Новая характеристика для позиции номенклатуры может быть добавлена и в момент оформления документа, например при поступлении запасов на склад.

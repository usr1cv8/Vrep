Справочник "Тарифные планы договоров обслуживания"

В программе может быть создано множество различных тарифных планов. Применение плана для того или иного клиента фиксируется в договоре с контрагентом (обязательное заполнение).

Список тарифных планов доступен из подраздела "Биллинг" раздела "Продажи".

В табличной части фиксируется список всех товаров, услуг и работ, которые попадут в формируемые по договору обслуживания документы.

Для продаваемых в кажом периоде (абонентских) позиций, у которых заполнено значение Включено в абонплату, цена товара/услуги может быть указана прямо в тарифном плане вручную.

Опция "Что делать при продаже внеплановых позиций?" позволит настроить ограничение на продажу номенклатурных позиций, не включенных в тарифный план, по данному договору.

Представление в счете

Для группировки в выставляемом счете нескольких номенклатурных позиций в одну, можно использовать колонку "Представление в счете", где выбрать другую номенклатуру — она и попадет в печатную форму. Данные преобразования можно будет увидеть в документе Счет на оплату, вкладка Расшифровка биллинга.

Дополнительно, можно включить автоматическое формирование представления номенклатуры, включающее в себя период оказания услуг (см. колонку Содержание* документов).

 

* Колонка Содержание включается в документах в Еще — Изменить форму.

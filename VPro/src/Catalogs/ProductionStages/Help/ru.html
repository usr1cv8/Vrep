Справочник "Этапы производства"

Справочник предназначен для хранения информации о этапах изготовления продукции, выполнения работ.

При использовании учета по нескольким подразделениям и включенной настройке программы "Этапы могут выполняться разными подразделениями", для каждого этапа можно указать перечень доступных структурных единиц, на которых возможно его выполнение, и указать значение по умолчанию.

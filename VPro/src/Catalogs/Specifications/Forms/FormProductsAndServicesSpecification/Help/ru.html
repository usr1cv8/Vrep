Список спецификаций номенклатуры

 

Существует несколько режимов отображения списка:

  • Все спецификации. Отображение всех возможных спецификаций номенклатуры
  • С характеристикой. Становится доступен список характеристик номенклатуры. Отображаются только спецификации выделенной в данный момент характеристики
  • Без характеристики. Отображаются только спецификации с неуказанной характеристикой

Дополнительно есть возможность отображения спецификаций, связанных с заказами покупателей

 

Над списком расположены кнопка выбора основной спецификации для связки номенклатура+характеристика и кнопки копирования спецификаций между разными элементами номенклатуры

 

Содержит следующую информацию о спецификациях:

  • Наименование. Наименование спецификации
  • Характеристика продукции. Отображается только в режиме просмотра всех спецификаций. Содержит значение, в случае привязки спецификации к конкретной характеристике.
  • Заказ покупателя. Заказ покупателя, к которому привязана спецификация. Колонка отображается только при включении режима просмотра спецификаций заказов
  • Комментарий. Произвольный комментарий к спецификации.
